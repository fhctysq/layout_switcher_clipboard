# layout_switcher_clipboard
custom layout switcher (Ukrainian ↔ English) and clipboard manager tool generated with Gemini

кастомний портативний буфер обміну з ізольованим копіюванням та зміною розкладки клавіатури. працює у фоновому режимі та споживає мінімум ресурсів.

* глобальна історія: тихо запам'ятовує всі копійовані тексти через Ctrl+C і Alt+C, зберігаючи до 512 записів.
* ізольований Alt-буфер: Alt+C — окреме копіювання, що не займає місце в системному буфері обміну (де лише 25 записів).
* збереження у файл: історія записується на диск із простим шифруванням. після закриття програми чи перезавантаження ПК записи не зникнуть.
* виправлення розкладки: забули перемкнути мову? виділіть текст та натисніть Pause.
* утиліта намагається використовувати не більше 1 МБ оперативної пам'яті. при досягненні ліміту старі записи звільняють ОЗП і залишаються лише у файлі.
* великі тексти: якщо текст понад 256 КБ, він відразу зберігається в окремий файл у папку ClipboardData поруч із програмою.

## гарячі клавіші (працює лише лівий Alt)

• Ctrl + C — звичайне копіювання (текст потрапляє і в систему, і в історію утиліти).
• Ctrl + V — звичайна вставка.
• Ctrl + A — виділяє текст, кастомний буфер копіює виділене.
• Alt + C — ізольоване копіювання (не затирає поточний Ctrl+C).
• Alt + V — ізольована вставка тексту, скопійованого через Alt+C (або найновішого, якщо не було Alt + C).
• Alt + B — відкриває віконце буфера. повторне натискання - навігація збереженими текстами донизу по одному (дублет клавіші ↓).
• Alt + N — навігація вгору по буферу (дублет клавіші ↑. якщо буфер закритий - відкриває віконце відразу на найстарішому записі).
• Pause — змінює розкладку виділеного тексту (Укр ↔ Англ).
• Alt + Pause — змінює малі літери на великі і навпаки (виправляє Caps Lock).
• Alt + X — вирізає текст лише в кастомний буфер.
• Alt + Z — дублює функцію скасування дії (Ctrl + Z).
• Alt + "-" (на нампаді) — з̶а̶к̶р̶е̶с̶л̶е̶н̶и̶й̶ ̶т̶е̶к̶с̶т̶
• Alt + "/" (на нампаді) — з̷а̷к̷р̷е̷с̷л̷е̷н̷и̷й̷ ̷т̷е̷к̷с̷т̷
• Alt + Q або Shift + хрестик на віконці — завершує роботу утиліти.

## керування в історії (Alt + B)

• клік лівою кнопкою миші по картці або Enter — вставляє текст у активне поле.
• клавіші ↑ та ↓ — навігація по списку (не працює, якщо текстове поле перехоплює стрілочки для системного буфера).
• Delete — видаляє вибраний запис з історії.
• Esc (або клік поза вікном) — закриває меню.

## компіляція (MSYS2 / MinGW) - вам потрібні лише файли layout_switcher_clipboard.cpp, resource.rc і resource.rc

```bash
windres resource.rc -o resource.o && g++ -Os -O2 -s -flto lsbuffer.cpp resource.o -o lsbuffer.exe -luser32 -lgdi32 -mwindows -luxtheme
```
