# layout_switcher_clipboard
custom layout switcher (Ukrainian ↔ English) and clipboard manager tool generated with Gemini

кастомний портативний буфер обміну з ізольованим копіюванням та зміною розкладки клавіатури. працює у фоновому режимі та споживає мінімум ресурсів.

* глобальна історія: тихо зберігає всі копійовані тексти через Ctrl+C і Alt+C, — до 512 записів
* ізольований Alt-буфер: Alt+C — окреме копіювання, що не займає місце в системному буфері обміну (де лише 25 записів)
* збереження у файл: історія записується на диск із простим шифруванням. після закриття програми чи перезавантаження ПК записи не зникнуть
* виправлення розкладки: забули перемкнути мову? виділіть текст та натисніть Pause
* утиліта намагається використовувати не більше 1024 КБ оперативної пам'яті у спокої. при досягненні ліміту старі записи звільняють ОЗП і залишаються лише у файлі
* великі тексти: якщо текст понад 256 КБ, він відразу зберігається в окремий файл у папку ClipboardData поруч із програмою

## гарячі клавіші (працює лише лівий Alt)

* Ctrl + C — звичайне копіювання (текст потрапляє і в системний буфер, і в історію утиліти)
* Ctrl + V — звичайна вставка (вставляє те, що скопійовано через Ctrl + C)
* Alt + Z - дублює Ctrl + Z
* Ctrl + A — виділяє текст як і повинно, але кастомний буфер копіює виділений текст
* Alt + C — ізольоване копіювання. Копіює текст у пам'ять утиліти, не затираючи ваш поточний текст на Ctrl + C
* Alt + X - вирізає лише в кастомний буфер
* Alt + V — ізольована вставка. вставляє текст скопійований через Alt + C (або останній з буфера, якщо не було Alt + C)
* Alt + Win + C - копіювання відразу в закріплені
* Alt + Win + X - вирізання в закріплені)
* Alt + Win + V - вставка останнього закріпленого
* Alt + B — відкриває віконце буфера. повторне натискання збереженими текстами по одному донизу
* Alt + N - навігує вгору по буферу (і якщо буфер не відкритий, то відкриває віконце відразу на останньому записі)
* Pause — змінює розкладку виділеного тексту (наприклад, qwerty замість йцукен) між латиницею і українською кирилицею. в обидва боки (Укр ↔ Англ)
* Alt + Pause - замінює малі літери на великі і навпаки (виправляє затиснутий Caps Lock)
* Alt + Q або Shift + хрестик на віконці - завершує роботу утиліти
* Alt + "-" (на нампаді) - з̶а̶к̶р̶е̶с̶л̶е̶н̶и̶й̶ ̶т̶е̶к̶с̶т̶
* Alt + "/" (на нампаді) - з̷а̷к̷р̷е̷с̷л̷е̷н̷и̷й̷ ̷т̷е̷к̷с̷т̷

## керування в історії (Alt + B)

* клік лівою клавішею миші по картці або Enter — вставляє текст у активне поле
* клік правою клавішею миші по картці — вибір без вставки
* клавіші ↑ та ↓ — навігація по списку (не працює, якщо текстове поле перехоплює стрілочки для системного буфера)
* Delete — видаляє вибраний запис з історії
* Esc (або клік поза вікном) — закриває меню

## компіляція (MSYS2 / MinGW) — вам потрібні лише файли layout_switcher_clipboard.cpp, resource.rc і resource.rc

```bash
windres resource.rc -o resource.o && g++ -Os -O2 -s -flto layout_switcher_clipboard.cpp resource.o -o layout_switcher_clipboard.exe -luser32 -lgdi32 -mwindows -luxtheme
```
